# Pyrenees Moto Trip

A 2-day motorcycle trip proposal through the Catalan Pyrenees, starting and ending in Barcelona.

**Two motorcycles, two couples, two days** — Barcelona to the Catalan Pyrenees and back, with an optional Andorra detour.

## View the proposal

👉 **[Live page](https://romain-rossi.github.io/barcelona-moto-trip/)**

## Route overview

- **Day 1:** Barcelona → Montserrat → Berguedà → Puigcerdà (~240 km)
- **Day 2:** Puigcerdà → Andorra → Col de Puymorens → La Seu d'Urgell → Barcelona (~310 km)
